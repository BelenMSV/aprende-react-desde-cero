# ⚛️ React Pro: De Cero a Experto

Bienvenido al repositorio oficial del curso **React Pro**. Este no es un curso de teoría superficial; es una ruta de aprendizaje diseñada bajo estándares de ingeniería de software reales, enfocada en crear aplicaciones escalables, mantenibles y performantes.

## 🚀 Presentación del Curso

Este curso ha sido estructurado para llevarte desde los cimientos de la librería hasta patrones de diseño avanzados utilizados en grandes empresas tecnológicas. Aprenderás no solo a escribir código de React, sino a **pensar en React**, dominando el ecosistema moderno (Vite, Tailwind, Hooks, Context, Testing y más).

---

## 📚 Estructura del Proyecto

El repositorio sigue una estructura estrictamente organizada para facilitar el aprendizaje autodidacta:

- **Módulos:** Carpetas numeradas (ej. `01-fundamentos-react`).
- **Temas:** Dentro de cada módulo, encontrarás subcarpetas por tema (ej. `01-introduccion-y-entorno`).
- **Documentación:** Cada tema contiene un `README.md` detallado con teoría, ejemplos de código y ejercicios.
- **Formato:** Los bloques de código están optimizados para ser copiados y probados directamente en tu entorno local.

---

## 🛠️ Conocimientos Previos Necesarios

Para aprovechar este curso al máximo, es recomendable que tengas bases sólidas en:

1.  **HTML5 & CSS3:** Semántica, Flexbox y Grid Layout.
2.  **JavaScript Moderno (ES6+):** - Arrow Functions.
    - Desestructuración de objetos y arreglos.
    - Operadores Spread y Rest.
    - Promesas y Async/Await.
    - Métodos de arreglos (`.map()`, `.filter()`, `.reduce()`).
3.  **Terminal:** Uso básico de comandos (`cd`, `ls`, `mkdir`).
4.  **Node.js:** Tener instalado Node (LTS) y conocer el uso básico de `npm` o `yarn`.

## 🗺️ Mapa de Ruta (Curriculum)

### [Módulo 01: Fundamentos y Filosofía de React](./01-fundamentos-react)
**⏱️ Duración estimada:** 12 - 15 horas.
En este módulo sentamos las bases. Aprenderás la diferencia entre el DOM Real y el Virtual, la sintaxis JSX, la creación de componentes funcionales, el paso de props y las mejores prácticas para estructurar tus proyectos y estilos. Es el pilar fundamental para todo lo que viene después.

### [Módulo 02: Estado y Ciclo de Vida (Hooks)](./02-estado-y-hooks)
**⏱️ Duración estimada:** 15 - 20 horas. (Próximamente)
Interactividad pura. Aprenderás a manejar el estado con `useState`, efectos secundarios con `useEffect` y a optimizar con `useRef`.

## ⚙️ Instalación y Uso Local

Si deseas probar los ejemplos de código de cada tema:

1.  **Clona el repositorio:**
    ```bash
    git clone [https://github.com/tu-usuario/react-pro-course.git](https://github.com/tu-usuario/react-pro-course.git)
    ```
2.  **Entra en la carpeta del tema que quieras practicar.**
3.  **Si hay un proyecto de Vite en su interior:**
    ```bash
    npm install
    npm run dev
    ```
